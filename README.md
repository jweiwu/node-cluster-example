# Node Cluster Example

## Usages

```bash
# copy and modify the .env
$ cp .env.example .env

# install dependencies
$ yarn

# serve at localhost:3000
$ yarn start

# hello world
$ curl http://localhost:3000/

# get the leader
$ curl http://localhost:3000/leader

# check is leader
$ curl http://localhost:3000/is-leader

```
